Useful resources

  • CodePen is a site where you can practice html, css, javascript before putting it on an official site
  • W3schools is a table of contents for all things web development
  • Lipsum Generator is a tool to randomly create paragraphs of placeholder text
  • mdn web docs is similar to W3 schools (a formal documentation of web developement) but different because it has more dense and "heavy" info
  • Adobe Color is a palette generator website which you can use to create color schemes which match with each other for the best design choices for your website
